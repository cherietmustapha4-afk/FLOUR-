# FLOU - Premium Social Experience ✨

[![React](https://img.shields.io/badge/React-18-blue.svg)](https://reactjs.org/)
[![Firebase](https://img.shields.io/badge/Firebase-10-orange.svg)](https://firebase.google.com/)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

A modern social media platform with real-time features, built with React and Firebase.

## ✨ Features

- 🔐 **Authentication** - Email/Password signup and login
- 📸 **Posts** - Create posts with multiple images, hashtags, and location
- ❤️ **Engagement** - Like and comment on posts with image support
- 👥 **Social** - Follow users, view followers/following
- 💬 **Real-time Chat** - Direct messaging with read receipts and typing indicators
- 🔍 **Search** - Search users, posts, and hashtags
- 🌙 **Dark Mode** - Toggle between light and dark themes
- 📍 **Location** - Add location to posts with interactive map
- 🔔 **Notifications** - Real-time notifications for likes, comments, and follows
- 🎨 **Premium UI** - Glassmorphism design with smooth animations

## 🚀 Tech Stack

- **Frontend**: React 18, React Hooks
- **Styling**: CSS3 with CSS Variables
- **Routing**: Hash-based routing
- **Backend**: Firebase (Authentication, Realtime Database, Storage)
- **Maps**: Leaflet + OpenStreetMap
- **Image Upload**: ImgBB API
- **UI Components**: Swiper.js, React-Leaflet
